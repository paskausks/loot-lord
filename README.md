# Loot Lord
A discord bot of sorts.

## Requirements

* Node.js 10+
* Yarn (https://yarnpkg.com)

## Setup

Install dependencies.

```
yarn install
```

## Running

Run tests

```
yarn test
```

Build sources

```
yarn build
```

Run the built files.

```
yarn start
```

Build sources and watch for changes

```
yarn build:watch
```

Run the linter

```
yarn lint
```

Run the linter and auto-fix issues

```
yarn lint:fix
```
